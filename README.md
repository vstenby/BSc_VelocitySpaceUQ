## Folder structure

In order for the code to work, you will need the following folder structure:

```bash
├── BSc_VelocitySpaceUQ
│   ├── README.md
│   ├── functions
│   ├── demos
│   ├── simulation
│   ├── images
│   ├── testprobs
│   └── other
│       └── tomoanalyticV
└── aux
    └── mosek
```

## Mosek installation

In order to get Mosek up and running, follow the installation guide here: https://docs.mosek.com/9.2/install/installation.html
